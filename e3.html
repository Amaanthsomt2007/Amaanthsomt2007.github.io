Project Description

Overview

PillowPop is a vibrant, playful landing page designed for a youthful audience. The project focuses on bright colors, friendly typography, and an energetic layout to create an engaging and fun shopping experience. It demonstrates versatility in frontend design by shifting from minimalist and luxury styles to a bold, colorful approach.

Features

Bright, engaging color palette

Friendly typography and playful branding

Clear call-to-action button with JavaScript feedback

Balanced hero layout with image and text

Feature section reinforcing product benefits

Technologies Used

HTML5 for clean page structure

CSS3 for colorful styling, Flexbox layout, and responsive design

JavaScript for user interaction feedback

How it Works

The hero section immediately establishes a fun tone through color and copy. Users can interact with the call-to-action button, which responds using JavaScript to simulate adding a product to a cart. The features section further highlights key selling points, while Flexbox ensures smooth alignment across devices.

Challenges Overcome

The main challenge was maintaining visual balance while working with bright colors and bold styling. I also addressed layout and spacing bugs to ensure the design remained clean and readable, especially on smaller screens.

What I Learned

This project reinforced my ability to troubleshoot UI glitches, fix responsiveness issues, and adapt design principles to different brand identities. I gained confidence in switching between design styles while maintaining functional, bug-free layouts.
