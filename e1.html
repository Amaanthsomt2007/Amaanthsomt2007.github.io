Project Description

Overview

SoftRest Pillows is a clean, modern affiliate-style landing page designed to promote comfort-focused sleep products. The project emphasizes clarity, trust, and simplicity, with a calm color palette and a balanced layout that guides users naturally from the headline to the call-to-action. The page is fully responsive and adapts smoothly across desktop and mobile devices.

Features

Clear hero section with persuasive headline and supporting text

Call-to-action button with JavaScript interaction feedback

Product-focused hero image for visual appeal

Feature highlights section to build trust and credibility

Mobile-responsive layout using Flexbox and media queries.

Technologies Used

HTML5 for semantic structure and content organization

CSS3 for layout, color styling, Flexbox positioning, and responsiveness

JavaScript for basic interactivity and user feedback

How it Works

The page introduces the product through a prominent hero section containing marketing copy and a call-to-action button. When the button is clicked, JavaScript provides instant feedback to simulate a shopping flow. Below the hero section, feature highlights reinforce product benefits and credibility. CSS Flexbox handles layout alignment, while media queries ensure usability on smaller screens.

Challenges Overcome

One challenge was ensuring the layout remained visually balanced on both desktop and mobile devices. I also resolved minor styling and spacing inconsistencies while refining button interaction and image presentation to maintain a clean, professional look.

What I Learned

This project strengthened my skills in debugging layout issues, fixing small UI glitches, and improving responsive behavior. I also gained better control over combining CSS Flexbox with JavaScript-driven interactions in a real-world landing page scenario.
